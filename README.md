## ⚙️ Programs Included

### 💳 SmartBank Transaction Engine

Simulates core banking operations including deposits, withdrawals, and balance validation using conditional logic.

### 👨‍💼 Employee Payroll Intelligence

Implements salary deduction logic based on employee attendance behaviour.

### 🎓 Academic Performance Analyzer

Evaluates student performance, calculates averages, and assigns grades using structured decision-making.

### 🔢 Number Classification System

Classifies numbers as positive, negative, or zero demonstrating fundamental conditional flow.

### 🧮 Factorial Computation Engine

Efficient factorial calculation using iterative algorithmic techniques.

### 🧩 Dynamic Pattern Generator

Generates numerical patterns dynamically using nested loops.
